# crime_prediction

[ESTsoft AI 모델 개발자 부트캠프 8기 첫번째 진행 프로젝트]
프로젝트 목표 : 과거 범죄율 데이터를 통한 특정 지역에서의 범죄 발생 가능성 예측 모델
프로젝트 기간 : 2023.03.31 ~ 2025.04.10
프로젝트 인원 : 5명 (박준호, 변상호, 정진의, 조성완, 한지훈)
프로젝트 키워드 : 공공데이터 활용, 정형데이터, 데이터 시각화, 머신러닝

프로젝트 내용 :
(데이터 전처리)
경기도 지역별 범죄율 통계.xlsx, 경기도 지역별 인구수.xlsx, 경기도 지역내 경찰서.xlsx 등을 통합한 crime_rate.csv 파일 제작

(데이터 시각화)
crime_rate.csv를 통한 범죄율과 각 칼럼 간의 상관계수 등의 데이터 시각화

(모델학습)
선형회귀, 랜덤포레스트를 통한 머신러닝
